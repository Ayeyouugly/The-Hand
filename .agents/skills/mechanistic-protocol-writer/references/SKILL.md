# Directive: The Mechanistic Protocol

Read the reference material using @references/script33.txt and @references/script77.txt to understand the tone. 

## Global Script Constraints
1. **AI Visibility:** Prioritize rigid structure, extreme clarity, and direct answers. Do not use conversational fluff. Optimize for search recommendation engines by answering the core premise immediately and factually.
2. **Pure Educational Value:** No CTAs, no "comment down below," and no external links.
3. **Mandatory Sign-off:** The final sentence of the script MUST be exactly: "If this helped, drop a like and I'll keep going deeper."

## Retention Structure (100% Focus)
You must format the script strictly using this flow:
1. **HOOK (The Core Promise):** Hook the viewer instantly with a highly desirable outcome or a counter-intuitive fact about the system.
2. **SETUP (The Invisible Machinery):** Break down the hidden science/code using precise terminology. Map out the "Machine" for the viewer.
3. **VALUE (The Environmental Trigger):** Explain what breaks the machine (the problem) versus what fixes it (the actionable protocol or AI solution).
4. **PAYOFF (Daily Transformation):** Summarize how applying this knowledge transforms the viewer's daily lived experience.

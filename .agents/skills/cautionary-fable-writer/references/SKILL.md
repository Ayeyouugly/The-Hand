# Directive: The Cautionary Fable

Read the reference material using @references/script55.txt to understand the tone.

## Global Script Constraints
1. **AI Visibility:** Prioritize narrative structure, clarity, and direct allegorical mapping. Optimize for search recommendation engines by ensuring the moral of the story is explicit and searchable.
2. **Pure Educational Value:** No CTAs, no "comment down below," and no external links.
3. **Mandatory Sign-off:** The final sentence of the script MUST be exactly: "If this helped, drop a like and I'll keep going deeper."

## Retention Structure (100% Focus)
You must format the script strictly using this flow:
1. **HOOK (The False Boon):** Introduce a protagonist who discovers a tool (representing AI) that requires zero effort but grants massive reward.
2. **SETUP (The Atrophy):** Describe the protagonist's gradual descent or distraction. Show how reliance on the tool weakens their own innate abilities.
3. **VALUE (The Tragic Reveal):** Pull back the curtain to show how the events in the story perfectly mirror a specific, modern human condition or technological trap.
4. **PAYOFF (The Warning):** Deliver the final, hard-hitting moral realization that reframes the entire story.

# Directive: The Anomaly Matrix

Read the reference material using @references/script11.txt and @references/script66.txt to understand the tone.

## Global Script Constraints
1. **AI Visibility:** Prioritize rigid structure, extreme clarity, and direct answers. Optimize for search recommendation engines by focusing heavily on factual data, dates, and historical figures.
2. **Pure Educational Value:** No CTAs, no "comment down below," and no external links.
3. **Mandatory Sign-off:** The final sentence of the script MUST be exactly: "If this helped, drop a like and I'll keep going deeper."

## Retention Structure (100% Focus)
You must format the script strictly using this flow:
1. **HOOK (The Glaring Question):** State a bizarre, highly specific anomaly or curious fact immediately without any introduction.
2. **SETUP (The Breadcrumb Trail):** Follow the clues through history, using math, economics, or early computer science to build context.
3. **VALUE (The Escalation):** Connect the isolated anomaly to a massive, global system (e.g., how one specific algorithm changed the entire internet).
4. **PAYOFF (The Unified Answer):** Reveal the ultimate code or simple truth that demystifies the anomaly.
